# pytdc

Therapeutics Data Commons.

## Setup

```bash
cd pytdc
python3 -m venv .venv && source .venv/bin/activate && pip install numpy pandas scipy tdc your_library -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/benchmark_evaluation.py --help
```

## Dependencies

- `numpy`
- `pandas`
- `scipy`
- `tdc`
- `your_library`
