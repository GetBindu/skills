# pymatgen

Materials science toolkit.

## Setup

```bash
cd pymatgen
python3 -m venv .venv && source .venv/bin/activate && pip install mp_api pymatgen requests yaml -q
```

## Environment variables

- `MP_API_KEY`

## Usage

```bash
python3 scripts/structure_analyzer.py --help
```

## Dependencies

- `mp_api`
- `pymatgen`
- `requests`
- `yaml`
