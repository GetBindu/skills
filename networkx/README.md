# networkx

Comprehensive toolkit for creating, analyzing, and visualizing complex networks and graphs in Python.

## Setup

```bash
cd networkx
python3 -m venv .venv && source .venv/bin/activate && pip install networkx numpy the -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/demo.py --help
```

## Dependencies

- `networkx`
- `numpy`
- `the`
