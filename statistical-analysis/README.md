# statistical-analysis

Guided statistical analysis with test selection and reporting.

## Setup

```bash
cd statistical-analysis
python3 -m venv .venv && source .venv/bin/activate && pip install matplotlib numpy pandas scipy seaborn -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/assumption_checks.py --help
```

## Dependencies

- `matplotlib`
- `numpy`
- `pandas`
- `scipy`
- `seaborn`
