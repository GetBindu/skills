# pymoo

Multi-objective optimization framework.

## Setup

```bash
cd pymoo
python3 -m venv .venv && source .venv/bin/activate && pip install matplotlib numpy pymoo -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/decision_making_example.py --help
```

## Dependencies

- `matplotlib`
- `numpy`
- `pymoo`
