# timesfm-forecasting

Google TimesFM 2.

## Setup

```bash
cd timesfm-forecasting
python3 -m venv .venv && source .venv/bin/activate && pip install importlib numpy pandas platform psutil timesfm torch -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/forecast_csv.py --help
```

## Dependencies

- `importlib`
- `numpy`
- `pandas`
- `platform`
- `psutil`
- `timesfm`
- `torch`
