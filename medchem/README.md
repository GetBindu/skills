# medchem

Medicinal chemistry filters.

## Setup

```bash
cd medchem
python3 -m venv .venv && source .venv/bin/activate && pip install datamol medchem pandas rdkit tqdm -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/filter_molecules.py --help
```

## Dependencies

- `datamol`
- `medchem`
- `pandas`
- `rdkit`
- `tqdm`
