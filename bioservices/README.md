# bioservices

Unified Python interface to 40+ bioinformatics services.

## Setup

```bash
cd bioservices
python3 -m venv .venv && source .venv/bin/activate && pip install bioservices multiple -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/compound_cross_reference.py --help
```

## Dependencies

- `bioservices`
- `multiple`
