# molfeat

Molecular ML featurization library (100+ featurizers: ECFP, descriptors, ChemBERTa).

## Setup

```bash
cd molfeat
python3 -m venv .venv && source .venv/bin/activate && pip install molfeat rdkit -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/molfeat_featurize.py --help
```

## Dependencies

- `molfeat`
- `rdkit`
