# Literature Review Writing Reference

## Four-Phase Workflow

### Phase 1: Reading Strategy
- Triage papers into P1 (must read), P2 (should read), P3 (skim)
- Create standardized note templates for each source
- P1: ~25% of papers, full read with detailed notes
- P2: ~50%, abstract + key sections
- P3: ~25%, abstract + conclusions

### Phase 2: Synthesis & Organization
- Build comparison tables across papers
- Create timeline analyses showing research evolution
- Develop taxonomy structures (categorize by multiple dimensions)

### Phase 3: Writing Structure
- Survey paper: Abstract → Intro → Background → Taxonomy → Analysis → Discussion → Future → Conclusion
- Thesis chapter: Intro → Literature Review → Critical Analysis → Research Gaps → Summary

### Phase 4: Writing Quality

**Citation Density Guidelines:**
| Section | Citations |
|---------|-----------|
| Introduction | 10-20 |
| Main Survey | 50-100+ |
| Discussion | 10-20 |
| Future Directions | 5-10 |

**Common Pitfalls:**
- List-style writing lacking analytical depth
- Missing connections between papers
- No clear taxonomy or organizing framework
- Insufficient critical analysis
- Weak transition phrases between ideas
