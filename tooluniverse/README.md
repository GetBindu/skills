# tooluniverse

Access 1000+ scientific tools from Harvard's ToolUniverse — bioinformatics, drug discovery, genomics, clinical research,

## Setup

```bash
cd tooluniverse
python3 -m venv .venv && source .venv/bin/activate && pip install tooluniverse -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/tooluniverse_search.py --help
```

## Dependencies

- `tooluniverse`
