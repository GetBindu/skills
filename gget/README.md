# gget

Fast CLI/Python queries to 20+ bioinformatics databases.

## Setup

```bash
cd gget
python3 -m venv .venv && source .venv/bin/activate && pip install gget pandas traceback -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/batch_sequence_analysis.py --help
```

## Dependencies

- `gget`
- `pandas`
- `traceback`
