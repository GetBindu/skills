# datamol

Python cheminformatics library (RDKit wrapper).

## Setup

```bash
cd datamol
python3 -m venv .venv && source .venv/bin/activate && pip install datamol selfies -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/datamol_process.py --help
```

## Dependencies

- `datamol`
- `selfies`
