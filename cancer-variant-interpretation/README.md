# cancer-variant-interpretation

ToolUniverse workflow — Cancer Variant Interpretation

## Setup

```bash
cd cancer-variant-interpretation
python3 -m venv .venv && source .venv/bin/activate && pip install tooluniverse -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/run.py --help
```

## Dependencies

- `tooluniverse`
