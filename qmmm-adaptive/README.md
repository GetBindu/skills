# qmmm-adaptive

QM/MM hybrid simulations with adaptive sampling for enzyme mechanisms and reaction dynamics.

## Setup

```bash
cd qmmm-adaptive
python3 -m venv .venv && source .venv/bin/activate && pip install ase numpy -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/qmmm_setup.py --help
```

## Dependencies

- `ase`
- `numpy`
