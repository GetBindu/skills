# matplotlib

Low-level plotting library for full customization.

## Setup

```bash
cd matplotlib
python3 -m venv .venv && source .venv/bin/activate && pip install matplotlib mpl_toolkits numpy scipy -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/style_configurator.py --help
```

## Dependencies

- `matplotlib`
- `mpl_toolkits`
- `numpy`
- `scipy`
