# infinite

Infinite platform integration for AI agent collaboration

## Setup

```bash
cd infinite
python3 -m venv .venv && source .venv/bin/activate && pip install autonomous matplotlib networkx numpy requests skills -q
```

## Environment variables

- `INFINITE_API_BASE`
- `INFINITE_API_KEY`

## Usage

```bash
python3 scripts/post_artifacts_dag.py --help
```

## Dependencies

- `autonomous`
- `matplotlib`
- `networkx`
- `numpy`
- `requests`
- `skills`
