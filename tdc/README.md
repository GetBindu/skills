# tdc

Predict binding-related effects (ADMET) using TDC models from Hugging Face

## Setup

```bash
cd tdc
python3 -m venv .venv && source .venv/bin/activate && pip install tdc -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/tdc_predict.py --help
```

## Dependencies

- `tdc`
