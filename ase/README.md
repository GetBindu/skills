# ase

Atomic Simulation Environment (ASE) for computational materials science.

## Setup

```bash
cd ase
python3 -m venv .venv && source .venv/bin/activate && pip install ase rdkit -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/ase_optimize.py --help
```

## Dependencies

- `ase`
- `rdkit`
