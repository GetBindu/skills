# pymc

Bayesian modeling with PyMC.

## Setup

```bash
cd pymc
python3 -m venv .venv && source .venv/bin/activate && pip install arviz matplotlib numpy pandas pymc scripts sklearn -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/model_comparison.py --help
```

## Dependencies

- `arviz`
- `matplotlib`
- `numpy`
- `pandas`
- `pymc`
- `scripts`
- `sklearn`
