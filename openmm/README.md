# openmm

OpenMM molecular dynamics engine for protein and ligand simulations.

## Setup

```bash
cd openmm
python3 -m venv .venv && source .venv/bin/activate && pip install openmm -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/openmm_md.py --help
```

## Dependencies

- `openmm`
