# datavis

Create scientific plots and visualizations using matplotlib and seaborn

## Setup

```bash
cd datavis
python3 -m venv .venv && source .venv/bin/activate && pip install matplotlib numpy pandas seaborn -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/plot_data.py --help
```

## Dependencies

- `matplotlib`
- `numpy`
- `pandas`
- `seaborn`
