# research-lookup

Look up current research information using Perplexity's Sonar Pro Search or Sonar Reasoning Pro models through OpenRouter.

## Setup

```bash
cd research-lookup
python3 -m venv .venv && source .venv/bin/activate && pip install requests research_lookup -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/research_lookup.py --help
```

## Dependencies

- `requests`
- `research_lookup`
