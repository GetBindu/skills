# chemical-compound-retrieval

ToolUniverse workflow — Chemical Compound Retrieval

## Setup

```bash
cd chemical-compound-retrieval
python3 -m venv .venv && source .venv/bin/activate && pip install tooluniverse -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/run.py --help
```

## Dependencies

- `tooluniverse`
