# citation-management

Comprehensive citation management for academic research.

## Setup

```bash
cd citation-management
python3 -m venv .venv && source .venv/bin/activate && pip install requests scholarly xml -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/extract_metadata.py --help
```

## Dependencies

- `requests`
- `scholarly`
- `xml`
