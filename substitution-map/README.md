# substitution-map

Map substitute materials for critical minerals with trade-off analysis and supply risk assessment

## Setup

```bash
cd substitution-map
python3 -m venv .venv && source .venv/bin/activate && pip install asyncio -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/substitution_lookup.py --help
```

## Dependencies

- `asyncio`
