# pydicom

Python library for working with DICOM (Digital Imaging and Communications in Medicine) files.

## Setup

```bash
cd pydicom
python3 -m venv .venv && source .venv/bin/activate && pip install numpy pydicom -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/anonymize_dicom.py --help
```

## Dependencies

- `numpy`
- `pydicom`
