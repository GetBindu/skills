# pointcloud-generator

Generate a colour-coded 3D point cloud (.

## Setup

```bash
cd pointcloud-generator
python3 -m venv .venv && source .venv/bin/activate && pip install matplotlib mpl_toolkits numpy -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/pointcloud_generator.py --help
```

## Dependencies

- `matplotlib`
- `mpl_toolkits`
- `numpy`
