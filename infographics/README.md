# infographics

Create professional infographics using Nano Banana Pro AI with smart iterative refinement.

## Setup

```bash
cd infographics
python3 -m venv .venv && source .venv/bin/activate && pip install dotenv requests -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/generate_infographic_ai.py --help
```

## Dependencies

- `dotenv`
- `requests`
