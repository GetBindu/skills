# rdkit

Cheminformatics toolkit for fine-grained molecular control.

## Setup

```bash
cd rdkit
python3 -m venv .venv && source .venv/bin/activate && pip install rdkit sascorer -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/rdkit_analysis.py --help
```

## Dependencies

- `rdkit`
- `sascorer`
