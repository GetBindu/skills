# alphafold-database

Access AlphaFold 200M+ AI-predicted protein structures.

## Setup

```bash
cd alphafold-database
python3 -m venv .venv && source .venv/bin/activate && pip install requests -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/_demo.py --help
```

## Dependencies

- `requests`
