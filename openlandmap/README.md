# openlandmap

Query ML-predicted global soil properties from EnvirometriX using different algorithms than SoilGrids

## Setup

```bash
cd openlandmap
python3 -m venv .venv && source .venv/bin/activate && pip install requests -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/query.py --help
```

## Dependencies

- `requests`
