# deepchem

Molecular ML with diverse featurizers and pre-built datasets.

## Setup

```bash
cd deepchem
python3 -m venv .venv && source .venv/bin/activate && pip install deepchem numpy traceback -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/graph_neural_network.py --help
```

## Dependencies

- `deepchem`
- `numpy`
- `traceback`
