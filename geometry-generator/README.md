# geometry-generator

Generate parametric bioinspired ribbed membrane STL geometry via LLM-guided design.

## Setup

```bash
cd geometry-generator
python3 -m venv .venv && source .venv/bin/activate && pip install core struct traceback -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/stl_generator.py --help
```

## Dependencies

- `core`
- `struct`
- `traceback`
