# stable-baselines3

Production-ready reinforcement learning algorithms (PPO, SAC, DQN, TD3, DDPG, A2C) with scikit-learn-like API.

## Setup

```bash
cd stable-baselines3
python3 -m venv .venv && source .venv/bin/activate && pip install gymnasium numpy stable_baselines3 -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/evaluate_agent.py --help
```

## Dependencies

- `gymnasium`
- `numpy`
- `stable_baselines3`
