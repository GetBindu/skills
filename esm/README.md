# esm

Comprehensive toolkit for protein language models including ESM3 (generative multimodal protein design across sequence, structure, and function) and ESM C (efficient protein embeddings and representations).

## Setup

```bash
cd esm
python3 -m venv .venv && source .venv/bin/activate && pip install torch transformers -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/demo.py --help
```

## Dependencies

- `torch`
- `transformers`
