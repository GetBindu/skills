# biorxiv-database

Efficient database search tool for bioRxiv preprint server.

## Setup

```bash
cd biorxiv-database
python3 -m venv .venv && source .venv/bin/activate && pip install requests -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/biorxiv_search.py --help
```

## Dependencies

- `requests`
