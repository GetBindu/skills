# torch-geometric

Graph Neural Networks (PyG).

## Setup

```bash
cd torch-geometric
python3 -m venv .venv && source .venv/bin/activate && pip install matplotlib networkx numpy scripts torch torch_geometric -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/visualize_graph.py --help
```

## Dependencies

- `matplotlib`
- `networkx`
- `numpy`
- `scripts`
- `torch`
- `torch_geometric`
