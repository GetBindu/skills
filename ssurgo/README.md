# ssurgo

Query detailed US soil survey data from USDA NRCS with ground-measured soil properties

## Setup

```bash
cd ssurgo
python3 -m venv .venv && source .venv/bin/activate && pip install requests -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/query.py --help
```

## Dependencies

- `requests`
