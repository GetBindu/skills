# protein-structure-retrieval

ToolUniverse workflow — Protein Structure Retrieval

## Setup

```bash
cd protein-structure-retrieval
python3 -m venv .venv && source .venv/bin/activate && pip install tooluniverse -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/run.py --help
```

## Dependencies

- `tooluniverse`
