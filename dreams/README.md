# dreams

Agentic materials discovery and DFT simulation framework using ASE, Quantum ESPRESSO, and Claude LLMs via LangGraph.

## Setup

```bash
cd dreams
python3 -m venv .venv && source .venv/bin/activate && pip install requests scientia -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/dreams_client.py --help
```

## Dependencies

- `requests`
- `scientia`
