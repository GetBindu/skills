# pydeseq2

Differential gene expression analysis (Python DESeq2).

## Setup

```bash
cd pydeseq2
python3 -m venv .venv && source .venv/bin/activate && pip install matplotlib numpy pandas pickle pydeseq2 -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/run_deseq2_analysis.py --help
```

## Dependencies

- `matplotlib`
- `numpy`
- `pandas`
- `pickle`
- `pydeseq2`
