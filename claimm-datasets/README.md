# claimm-datasets

Search and discover NETL EDX CLAIMM datasets with 200+ US-focused critical minerals datasets

## Setup

```bash
cd claimm-datasets
python3 -m venv .venv && source .venv/bin/activate && pip install asyncio cmm_data -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/claimm_search.py --help
```

## Dependencies

- `asyncio`
- `cmm_data`
