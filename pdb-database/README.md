# pdb-database

Python API for RCSB PDB 3D structures (search, fetch coordinates, metadata).

## Setup

```bash
cd pdb-database
python3 -m venv .venv && source .venv/bin/activate && pip install requests -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/query.py --help
```

## Dependencies

- `requests`
