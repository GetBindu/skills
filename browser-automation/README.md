# browser-automation

Playwright-based browser automation for scraping JavaScript-rendered scientific databases

## Setup

```bash
cd browser-automation
python3 -m venv .venv && source .venv/bin/activate && pip install playwright requests -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/browser_fetch.py --help
```

## Dependencies

- `playwright`
- `requests`
