# gwas-drug-discovery

ToolUniverse workflow — Gwas Drug Discovery

## Setup

```bash
cd gwas-drug-discovery
python3 -m venv .venv && source .venv/bin/activate && pip install tooluniverse -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/run.py --help
```

## Dependencies

- `tooluniverse`
