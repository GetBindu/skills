# symtorch

Approximate deep learning model components with symbolic equations using PySR

## Setup

```bash
cd symtorch
python3 -m venv .venv && source .venv/bin/activate && pip install requests scientia -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/symtorch_client.py --help
```

## Dependencies

- `requests`
- `scientia`
