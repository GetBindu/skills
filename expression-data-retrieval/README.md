# expression-data-retrieval

ToolUniverse workflow — Expression Data Retrieval

## Setup

```bash
cd expression-data-retrieval
python3 -m venv .venv && source .venv/bin/activate && pip install tooluniverse -q
```

## Environment variables

None.

## Usage

```bash
python3 scripts/run.py --help
```

## Dependencies

- `tooluniverse`
